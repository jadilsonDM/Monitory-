1 - Página inicial

Esta página funciona como página inicial, nela consiste um botão de atalho para o socorro denominado EMERGÊNCIA , ao clicar neste botão, o usuário será direcionado automaticamente para a página Socorro onde fará a solicitação dos socorristas.

Assim como as demais páginas, agrupa o menu de páginas de todo o site em seu cabeçalho.

A Página inicial contém informativos atualizados para cuidados e saúde além de um texto de apresentação do aplicativo.

2 - Página de Socorro

Nesta página o usuário terá todas as opções de EMERGÊNCIA disponíveis para que possam ser atendidas pelos socorristas.

Acidente: Ao clicar neste botão, o usuário será direcionado para a página de socorro onde estará relatando breves dados relacionados ao acidente.

Elevador: Ao clicar neste botão, o usuário será direcionado para a página de socorro onde estará relatando breves dados relacionados a problemas no elevador.

Incêndio: Ao clicar neste botão, o usuário será direcionado para a página de socorro onde estará relatando breves dados relacionados ao incêndio.

Mal Súbito: Ao clicar neste botão, o usuário será direcionado para a página de socorro onde estará relatando breves dados relacionados a mal súbito.

Energia: Ao clicar neste botão, o usuário será direcionado para a página de socorro onde estará relatando breves dados relacionados a queda de energia.

2.1 - Acidente

Esta página exibe o preenchimento de informações necessárias para o socorro, onde os socorristas estarão cientes de quem está chamando o socorro. Ao preencher o nome e matrícula em Dados do Necessitado o usuário estará informando seus dados, marcando as opções de sexo e idade com apenas uma alternativa.

Local de Emergência: Nessa caixa o usuário estará registrando o bloco, andar e locais específicos selecionando uma das opções em cada caixa de escolha, ainda poderá deixar observações adicionais na caixa OBS caso queira acrescentar informações que não constam na página.

Informações adicionais: Nessa caixa o usuário irá adiantar informações para que os socorristas possam ser encaminhados para o local com os devidos tipos de equipamentos baseados nos dados informados pelo usuário através marcações em Suspeitas de e Membros afetados.

Clicando no botão ENVIAR o usuário estará encaminhando para os socorristas todas as informações registradas por ele na página.
