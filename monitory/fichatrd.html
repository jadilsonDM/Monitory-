1 - Home

This page acts as the home page, it consists of a shortcut button for help called EMERGENCY, when clicking this button, the user will be automatically directed to the Help page where he will make the request of the first responders.

Like the other pages, it groups the menu page of the entire site in its header.

The Home page contains updated newsletters for care and health as well as text introducing the app.

2 - Help page

On this page the user will have all the EMERGENCY options available so that they can be attended by rescuers.

Accident: When clicking on this button, the user will be directed to the help page where he will be reporting brief data related to the accident.

Elevator: When clicking on this button, the user will be directed to the help page where he will be reporting brief data related to problems with the elevator.

Fire: When clicking on this button, the user will be directed to the help page where they will be reporting brief data related to the fire.

Sudden Illness: When clicking on this button, the user will be directed to the help page where he will be reporting brief data related to sudden illnes.

Energy: By clicking on this button, the user will be directed to the help page where he will be reporting brief data related to the power outage.

2.1 - Accident

This page displays the required information for the rescue, where the rescuers will be aware of who is calling the rescue. When filling in the name and registration in Data of the Needy, the user will be informing his data, marking the sex and age options with only one alternative.

Emergency Location: In this box the user will be registering the block, floor and specific locations by selecting one of the options in each choice box, he can still leave additional comments in the OBS box if he wants to add information that is not on the page.

Additional information: In this box, the user will provide information so that rescuers can be directed to the site with the appropriate types of equipment based on the data provided by the user through the Suspected and Affected Members markings.

By clicking on the SEND button the user will be forwarding to the rescuers all the information recorded by him on the page.
